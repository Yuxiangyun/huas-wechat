# Open Source Checklist

这个仓库已经完成一轮基础脱敏，当前重点变更包括：

- 管理员 Basic Auth 改为环境变量
- 上游学校域名改为环境变量/占位值
- 微信小程序 `appid` 改为 `touristappid`
- 运行时 `data/` 目录改为默认忽略提交
- 公开页面里的个人联系方式、公众号配置、贡献者实名信息改为占位内容

发布前仍建议你再手动确认以下项目：

- `huas-weixin/miniprogram/assets/campus-map/` 里的地图素材仍然是业务素材，如果你不希望公开，请自行替换
- 目录名、脚本名和部分文档标题仍保留 `huas` 命名；如果你想做成完全中性的模板，需要继续统一重命名
- `huas-weixin/doc/html:css/` 是一份静态文档快照，如果准备公开，建议确认里面的展示文案是否仍符合你的预期
- 本地若已有 `.env`、数据库、上传图片、开发者私有配置，请确认没有被手工强制提交
