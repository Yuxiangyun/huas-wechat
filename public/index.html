Cache Behavior
Backend: 第 1 次 ?refresh=true → 跳过缓存, upstream() 请求教务 → CacheService.set() 原子 upsert 写入 SQLite
第 2 次无 refresh → CacheService.get() 命中缓存 → _meta.cached=true
第 3 次 ?refresh=true → 再次请求上游覆盖缓存

三步：refresh → 命中缓存 → 再次 refresh
